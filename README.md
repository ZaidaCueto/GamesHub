### 🎲 Para testar o projeto

```bash
# Clone este repositório
$ git clone https://github.com/ZaidaCueto/HubGames.git

# Acesse a pasta do projeto no terminal/cmd
$ cd HubGames

# Para rodar o projeto   execute
$ dotnet run
```
